# IEEE LaTeX Starter

This folder contains a submission-friendly IEEE manuscript starter mapped from the current project draft.

## Files

- `main.tex`: Primary manuscript file (IEEE conference format).
- `references.bib`: BibTeX reference placeholders to replace with real sources.

## Build Locally

Use any LaTeX workflow that supports IEEEtran and BibTeX.

Example commands:

1. `pdflatex main.tex`
2. `bibtex main`
3. `pdflatex main.tex`
4. `pdflatex main.tex`

## What To Replace Before Submission

1. Replace placeholder email in `main.tex`.
2. Replace all entries in `references.bib` with real papers and valid metadata.
3. Replace placeholder screenshots in `docs/images/` with final captures.
4. Fill validation table values in `main.tex`.
5. Add final measured results from physical testing.
